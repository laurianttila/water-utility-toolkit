# Sociotechnical Diagnosis Canvases
## Englsih versions will be added durin Q1 2026
A set of three A2-sized workshop templates used to facilitate sociotechnical mapping of the utility and its perceived threat landscape. 
* **Sociotechnical Lens**: Treats the utility as an inseparable mix of technology and people.
* **Deep-Dive Analysis**: Helps non-experts identify high-priority risks often missed by purely technical lenses.
